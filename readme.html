1. Clone the repository into a directory of your choice. 2. In your terminal, change into said directory and run docker-compose up -d 3. Dev server runs on port 8001, phpmyadmin on port 8081 4. go to http://localhost:8001 in your browser. 5. You should be able to install wordpress and create a username and password. 6. Next login to the wordpress site. 7. in the dashboard's sidebar click on "Pages" 8. Click on Sample Page 9. On sample page find the right sidebar and click the dropdown for page attributes. 10. click on templates and select the template to use. 11. Hit update at the at the top of screen and then hit preview button. 12. One template has the base html and content, and the other is the fetching function.
